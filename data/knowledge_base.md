# База знань для AI-консультанта

## 1. Щорічна відпустка

Працівник може використати щорічну оплачувану відпустку після 6 місяців безперервної роботи у компанії.

Стандартна тривалість щорічної основної відпустки становить 24 календарні дні.

Заяву на щорічну відпустку потрібно подати не пізніше ніж за 14 календарних днів до запланованої дати початку відпустки.

Якщо працівник має невикористані дні відпустки, вони можуть бути перенесені на наступний період.

У цій базі знань не описані винятки для окремих категорій працівників.

## 2. Sick leave policy

Sick leave must be confirmed by an official medical certificate.

The employer must register the sick leave record before payment processing.

If the medical certificate is missing, the absence cannot be automatically treated as paid sick leave.

The knowledge base does not contain information about the exact amount of sick leave compensation.

The knowledge base does not contain rules for calculating sick leave payment.

## 3. Індексація зарплати / Salary indexation

Індексація зарплати застосовується, якщо індекс споживчих цін перевищив встановлений поріг.

Salary indexation should not be applied automatically without checking the current indexation base month.

For payroll systems, the base month must be stored separately for each employee.

Для розрахунку індексації потрібні додаткові дані: базовий місяць працівника, актуальний індекс споживчих цін та сума доходу, яка підлягає індексації.

У цій базі знань немає достатніх числових даних для розрахунку конкретної суми індексації.

## 4. Податкові строки

ЄСВ за найманих працівників сплачується у строки, встановлені для відповідного звітного періоду.

Якщо останній день строку припадає на вихідний або святковий день, строк може переноситися на найближчий робочий день.

У цій базі знань точні календарні дати сплати ЄСВ не наведені.

AI-консультант не має називати точну дату сплати ЄСВ, якщо вона відсутня у джерелі.

## 5. Документи при прийнятті на роботу

Перед початком роботи працівник має надати роботодавцю документи, необхідні для оформлення трудових відносин.

До базового переліку можуть входити паспорт або інший документ, що посвідчує особу, реєстраційний номер облікової картки платника податків та документи про освіту або кваліфікацію, якщо це потрібно для посади.

Трудовий договір або наказ про прийняття на роботу має бути оформлений до фактичного допуску працівника до роботи.

У цій базі знань не наведено повний перелік документів для спеціальних категорій працівників.

## 6. Випробувальний строк

Випробувальний строк може встановлюватися під час прийняття працівника на роботу, якщо це передбачено умовами трудового договору або наказом про прийняття.

Мета випробувального строку — перевірити відповідність працівника роботі, на яку його приймають.

У цій базі знань не наведено максимальної тривалості випробувального строку для різних категорій працівників.

AI-консультант не має вигадувати конкретну тривалість випробувального строку, якщо вона не вказана у джерелі.

## 7. Remote work policy

Remote work may be used when the employee and employer agree on the remote work format.

The company should define how tasks are assigned, how work results are reported, and which communication channels are used.

Remote work does not automatically cancel the employee's obligation to follow internal company policies.

The knowledge base does not contain rules about compensation for internet, electricity, or home office equipment.

## 8. Звільнення за власним бажанням

Працівник може ініціювати звільнення за власним бажанням, подавши відповідну заяву роботодавцю.

Заява має містити бажану дату припинення трудових відносин.

Роботодавець має оформити припинення трудових відносин відповідним наказом.

У цій базі знань не наведено точний строк попередження роботодавця про звільнення.

AI-консультант не має вигадувати строк попередження, якщо він відсутній у джерелі.

## 9. Document processing rules

When processing documents, the system should extract only facts that are explicitly present in the source text.

If the source document contains a general rule but does not contain exact numbers, dates, exceptions, or formulas, the system must not invent them.

If a user asks for a calculation and the source does not provide all required numeric values, the system should explain that the calculation cannot be completed based only on the provided knowledge base.

If several sections are relevant, the answer should mention the limitation of the available context.

## 10. Правила відповіді AI-консультанта

AI-консультант має відповідати українською мовою.

AI-консультант може використовувати англомовні фрагменти контексту, але фінальна відповідь користувачу має бути українською мовою.

Відповідь має базуватися тільки на наданій базі знань.

Якщо інформації недостатньо, AI-консультант має прямо сказати, що в базі знань недостатньо даних для точної відповіді.

AI-консультант не має вигадувати точні дати, суми, формули, винятки або юридичні норми, якщо вони відсутні в базі знань.

AI-консультант має вказувати, на які секції бази знань спирається відповідь.
